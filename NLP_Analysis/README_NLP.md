# 🧠 Natural Language Processing (NLP) & Sentiment Engine

This directory contains the downstream analytical notebooks that process the structured datasets generated by the Web Scraping ETL pipelines.

## ⚙️ Execution Order
To ensure full reproducibility, the analytical notebooks in this folder must be executed **strictly after** the data extraction phase:
1. Run the target scraper from the `/Scrapers/` directory (e.g., `bolivia_24h_tracker.py` or the Ecuador `main_orchestrator.py`).
2. Verify that the outputs have been correctly saved in the `/Data/Raw/` and processed into the `/Data/Cleaned/` directory.
3. Open the Jupyter Notebooks (`.ipynb`) in this folder. Update the data paths in the first cell to point to your local `/Data/Cleaned/` folder.

## 📦 Core NLP Dependencies
The notebooks rely on advanced Transformer-based models for Spanish language processing. Ensure you have the following environment setup:
- `pysentimiento` (RoBERTuito-base for sentiment & emotion analysis in Spanish).
- `wordcloud`, `matplotlib`, `seaborn` (For Data Visualization).
- `nltk` or `spaCy` (For tokenization and NER - Named Entity Recognition).

> **Note:** Execution in Google Colab (with T4 GPU enabled) is highly recommended for processing thousands of news headlines through the RoBERTuito pipeline without bottlenecking local CPU resources.
